# PS5 Media Renamer
A Python utility that automatically organizes and renames PS5 capture files (videos and screenshots) with clean, sequential filenames. Scans your PS5 media folders and renames files based on their game directory, making it easier to manage and share your gaming content.


### Build EXE
`python -m PyInstaller --onefile --console --name "ps5_media_renamer" --clean ps5_media_renamer.py`
